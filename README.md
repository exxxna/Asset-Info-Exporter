# Asset Info Exporter
BOOTHショップ向けのアセット情報出力ツールです。

## 概要

参照するPrefabと商品フォルダを指定することで、以下の情報を取得して出力します。

- 前提アセット
- ポリゴン数
- マテリアル数
- テクスチャメモリ容量
- PBコンポーネント数
- PBコライダー数
- パラメーター容量
- フォルダ構成（ファイルは拡張子ごとに合計数表記）
